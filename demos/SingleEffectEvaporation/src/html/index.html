Directions

This demonstration shows the efficacy of single-effect evaporation of a dilute sugar solution. The saturated or unsaturated liquid is fed into a tank, which contains a steam-fed heat exchanger. The user may adjust the feed flow rate and feed temperature of the feed solution. The conditions of each inlet and outlet stream are shown in colored boxes, with green boxes representing the sugar solution or vapor from the sugar solution, and red boxes representing the steam fed into the heat exchanger.

Details

Assuming that the vapor in the hot-side of the heat exchanger is saturated, the equation for the heat rate of a heat exchanger in a single-effect evaporator is:

$$ [1] \quad \dot{ Q } = U A ( T_{ h } - T_{ conc } ) $$

where \( \dot{ Q } \) is heat rate (W), \( U \) is the overall heat transfer coefficient of the heat exchanger [W/(m2K)], \( A \) is the area of heat transfer of the heat exchanger (m2), \( T_{ conc } \) is the temperature of the concentrate in the evaporator (K), and \( T_{ h } \) is the temperature of the vapor in the heat exchanger (K), which is typically saturated steam. The value of \( U \) depends on the type of fluid on each side of the heat exchanger, because the fluid's density, viscosity, velocity, thermal conductivity, and heat capacity affect the value of \( U \). First, one can determine the values of \( T_{ h } \) and \( T_{ conc } \) using Antoine's equation, assuming the pressure of the evaporator and heat exchanger are known:

$$ [2] \qquad \mathrm{ log }_{ 10 } \left( P \right) = A - \frac{ B }{ T + C } $$

where \( P \) is the pressure in the evaporator or heat exchanger (Pa), \( A \), \( B \), and \( C \) are constants, and \( T \) is the temperature (K) in the evaporator or heat exchanger. The Antoine's Equation constants used in this simulation are available from reference (1). For a first approximation, one can assume that the concentrate is a single component, e.g. pure water in this simulation. This will allow us to determine an initial value for \( T_{ conc } \). A more accurate method to determine \( T_{ conc } \) will be discussed later. Once the hot and cold-side temperatures have been determined, one can find the amount of steam required using the following equation:

$$ [3] \quad F_{ h } = \frac{ \dot{ Q } }{ \Delta H_{ vap, h } } $$

where \( F_{ h } \) is the mass flow rate (kg/s) and \( \Delta H_{ vap, h } \) is the heat of vaporization (J/kg) of the hot-side fluid, which is steam in this simulation. The values for \( \Delta H_{ vap } \) of water in this simulation were determined using an approximation from reference (2). The rate of water evaporation (kg/s) of the solution in the evaporator is determined using the following equation:

$$ [4] \quad F_{ evap } = \frac{ \dot{ Q } - F_{ in } \, C_{ p, c } \, \left( T_{ conc } - T_{ in } \right) }{ \Delta H_{ vap, c } } $$

where \( F_{ evap } \) is the mass flow rate of evaporated vapor (kg/s), \( F_{ in } \) is the inlet flow rate of the solution, \( C_{ p, c } \) is the heat capacity of the solution [J/(kg K)], \( T_{ in } \) is the temperature of the inlet liquid, and \( \Delta H_{ vap, c } \) is the heat of vaporization of the solution in the evaporator. In this simulation, \( \Delta H_{ vap, c } \) was determined from reference (1) and \( C_{ p, c } \) was determined from reference (3). Flow rate of the concentrate (kg/s) is determined using a mass balance:

$$ [5] \quad F_{ conc } = F_{ in } - F_{ evap } $$

where \( F_{ conc } \) is the flow rate of concentrate out of the bottom of the evaporator. The mass fraction of solute in the concentrate can also be determined using a mass balance:

$$ [6] \quad x_{ conc } = \frac{ x_{ in } \, F_{ in } }{ F_{ conc } } $$

where \( x_{ conc } \) is the mass fraction of solute in the concentrate stream (kg/kg) and \( x_{ in } \) is the mass fraction of solute in the inlet stream. The steam economy (kg/kg) is the ratio of the mass of liquid evaporated to the mass of steam used:

$$ [7] \quad SE = \frac{ F_{ evap } }{ F_{ h } } $$

Equations [1] - [6] can be used to obtain a first-approximation of the concentration of solute in the concentrate stream and the flow rate of each stream. However, we assumed in equations [1] and [2] that the concentrate is a single component, e.g. pure water. Using the mass fraction of solute (\( x_{ conc } \)) calculated from equation [6], one can determine a more accurate temperature of the concentrate using Raoult's law. First, calculate the saturation pressure of the concentrate:

$$ [8] \quad P_{ sat, conc } = \frac{ P_{ evap } }{ 1 - x_{ conc } } $$

where \( P_{ sat, conc } \) is the saturation pressure of the concentrate and \( P_{ evap } \) is the pressure inside the evaporator. Next, use \( P_{ sat, conc } \) to solve Antoine's Equation [2] for \( T_{ conc } \). Finally, recalculate equations [1] and [3] - [7] using the updated temperature of the concentrate (\( T_{ conc } \)). Repeat until the desired precision is achieved.
