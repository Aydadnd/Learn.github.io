Directions

This simulation generates a vector field around a sphere with radius \( r = 1 \; \mathrm{ m } \). One of three fluids can be selected from the dropdown menu. The velocity of the fluid can be adjusted with the slider below the dropdown menu. The vector field will automaticaly update if a parameter is changed. Note that the velocity of the fluid may change automatically when a new fluid is selected, in order to keep the Reynold's number within a valid domain (because the governing equations are only valid for a certain range of Reynold's numbers). Click "Details" to see the governing equations used in this simulation.

Details

The Navier-Stokes equations are a set of partial differential equations that describe the motion of viscous fluids. Mathematically, they express conservation of momentum and mass1. Few analytical solutions exist for the Navier-Stokes equations, so numerical approximations are often used to model fluid behavior. Assuming incompressible fluid, steady-state flow, and radial symmetry about the x-axis (which goes left-to-right in this simulation), the Navier-Stokes equation of motion in spherical coordinates is2

$$ - \frac{1}{ r^{2} \; \mathrm{sin} ( \theta ) } \frac{ \partial ( \Psi , D^{ 2 } \Psi ) }{ \partial ( r, \theta ) } + \frac{ 2 }{ r^{ 2 } } \frac{ D^{ 2 } \Psi }{ \mathrm{ sin }^{ 2 } ( \theta ) } \left( \frac{ \partial \Psi }{ \partial r } \mathrm{ cos }( \theta ) - \frac{ 1 }{ r } \frac{ \partial \Psi }{ \partial \theta } \mathrm{ sin } ( \theta ) \right) - \frac{ 2 }{ Re } D^{ 4 } \Psi = 0 $$

where \( r \) is radius, \( \theta \) is the polar angle, \( \Psi \) is the stream function, \( Re \) is the Reynolds number and \( D \) is

$$ D^{ 2 } = \frac{ \partial^{ 2 } }{ \partial r^{ 2 } } + \frac{ \mathrm{ sin }( \theta ) }{ r^{ 2 } } \frac{ \partial }{ \partial \theta } \left( \frac{ 1 }{ \mathrm{ sin }( \theta ) } \frac{ \partial }{ \partial \theta } \right) $$

The Reynolds number is

$$ Re = \frac{ \rho V D }{ \mu } $$

where \( \rho \) is the density of the fluid, \( V \) is the magnitude of the fluid velocity vector, \( D \) is the characteristic length, and \( \mu \) is the dynamic vicosity of the fluid. The x-component \( u \) and y-component \( v \) of the velocity vector \( \textbf{ V } \) are defined

$$ u = \frac{ 1 }{ r^{ 2 } \; \mathrm{ sin } \theta } \frac{ \partial \Psi }{ \partial \theta } $$ $$ v = - \frac{ 1 }{ r \; \mathrm{ sin } \theta } \frac{ \partial \Psi }{ \partial r } $$

The boundary conditions for flow around a sphere are

$$ \Psi = \frac{ 1 }{ 2 } r^{ 2 } \mathrm{ sin }^{ 2 } \theta, \; \; r \rightarrow \infty $$ $$ \Psi = \frac{ \partial \Psi }{ \partial r } = 0, \; \; r = R $$

where \( R \) is the radius of the sphere. The numerical "Galerkin method" used in this simulation approximates the stream function \( \Psi \) as a polynomial in which coefficients are determined experimentally. For this reason, the solution is only valid for low Reynolds numbers \( 0 \leq Re \leq 500 \).
